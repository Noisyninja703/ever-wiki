# Linux Tips

<h3>Navigate in terminal</h3>

```
cd - {Go to previous Directory}
cd ~ {Go to root}
mkdir <dir name> && cd <dir name> {Make new directory and cd into it}
touch <Filename> {Make a new file with name and type}
nano <Filename> {Edit file in terminal}
```

<h3>Nano Controls</h3>

```
CTRL + O --> Save file
CTRL + X --> Exit Nano
```
